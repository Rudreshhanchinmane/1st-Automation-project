Project Title

Smart Fan/AC Automation and Protection System using VFD, Relays, and Motor Starters

Project Description

This project demonstrates an automatic control and protection system for a fan or AC unit using VFD, contactors, relays, and protection devices.

  • Automatic & Manual operation using push buttons and control logic
  • Controlled via Danfoss VLT LCP 31 for efficient motor operation
  • LC1D contactors used for ON/OFF switching
  • MCBs, thermal overloads provide system protection
  • Signal lamps indicate system status
  • Neat wiring with terminal blocks, connectors, and ducts

Day 1 Work

On the first day, we built the base panel structure, installed terminal blocks, MCBs, and designed the panel layout for wiring. Started connection of relays and buttons.
